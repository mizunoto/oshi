# 推し度チェッカー

HTML5 + CSS3 + JavaScript だけで動作する、推しキャラの推し度を比較してランキング化するローカルアプリです。データは `localStorage` に保存されます。

## 主な機能

- 推しキャラの追加（カンマ区切りで複数追加も可能）
- 左 / 決められない / 右の 3 択投票で推し度を更新
- 推し度ランキングの自動生成（勝敗・引き分けも表示）
- キャラクターの削除・データのリセット

## 使い方

1. `index.html` をブラウザで開きます。
2. 推しキャラ名を入力して「追加」を押します。
3. 左右のカードから好きな方、または「決められない」を選んで投票します。
4. ランキングが自動更新されます。

## データ保存について

ランキングや戦績はブラウザの `localStorage` に保存されます。別の端末やブラウザでは共有されません。
